For $x \geq 3$: $|x-1| + |x-3| = (x-1) + (x-3) = 2x-4 = 2$, so $x = 3$. Therefore, all $x$ with $1 \leq x \leq 3$ are solutions, giving infinitely many integer solutions.</p>
</details>
