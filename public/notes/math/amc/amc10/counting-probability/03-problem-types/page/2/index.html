🧩 Problem Types#

A comprehensive catalog of common AMC counting and probability problem patterns with recognition cues and solution templates.

📚 Problem Type Overview#

🔢 Basic Counting Patterns#

  • Seating & Restrictions - Adjacency and spacing problems
  • Word Rearrangements - Letter arrangements with constraints
  • Committees & Conditions - Selection with roles and constraints
  • Balls in Bins - Integer solutions and distribution problems

🎯 Advanced Counting Patterns#

  • At Least/At Most - Complement and inclusion-exclusion strategies
  • Grid Paths - Lattice walks with restrictions
  • Circular Permutations - Ring arrangements and reflections
  • Derangements - No element in original position

🎲 Probability Patterns#

  • Urn, Coin & String Problems - Replacement vs. no-replacement
  • Expected Counts - Indicator variables and linearity
  • Conditional Chains - Total probability and Bayes
  • Symmetry Probability - Random choices via symmetry

🔍 Specialized Patterns#

  • Hypergeo vs Binomial - Distribution recognition
  • Digit Counting - Divisibility and base representations

🎯 How to Use This Section#

  1. Recognition: Learn the cues that identify each problem type
  2. Template: Follow the solution template for each type
  3. Practice: Work through the AMC-style examples
  4. Pitfalls: Study common mistakes and how to avoid them
