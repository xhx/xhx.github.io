🎯 Essential Divisibility Rules#

Basic Rules (Memorize These!)#

Divisor Rule Example Explanation

🔍 How to Apply Divisibility Rules#

Step-by-Step Process#

  1. Identify the divisor - What number are you checking divisibility by?
  2. Choose the appropriate rule - Use the most efficient rule for that divisor
  3. Apply the rule - Follow the specific procedure
  4. Check your work - Verify the result

Example Problems#

Example 1: Check if 123456 is divisible by 3#

Solution: Sum of digits = 1 + 2 + 3 + 4 + 5 + 6 = 21 Since 21 ÷ 3 = 7, the number is divisible by 3.

Example 2: Check if 123456 is divisible by 4#

Solution: Last 2 digits = 56 Since 56 ÷ 4 = 14, the number is divisible by 4.

Example 3: Check if 123456 is divisible by 11#

Solution: Alternating sum = 1 - 2 + 3 - 4 + 5 - 6 = -3 Since -3 is not divisible by 11, the number is not divisible by 11.

🧠 Mental Math Shortcuts#

Quick Recognition Techniques#

  1. For 3 and 9: Sum of digits

    • 123: 1+2+3 = 6 (divisible by 3, not 9)
    • 1233: 1+2+3+3 = 9 (divisible by both 3 and 9)

  2. For 4: Last 2 digits

    • 1236: 36 ÷ 4 = 9 ✓
    • 1234: 34 ÷ 4 = 8.5 ✗

  3. For 11: Alternating sum

    • 121: 1-2+1 = 0 ✓
    • 123: 1-2+3 = 2 ✗

  4. For 6: Check both 2 and 3

    • 1236: Even ✓, sum = 12 (divisible by 3) ✓

🎯 Common AMC Problem Types#

Type 1: Find the largest/smallest number with given divisibility#

Strategy: Use divisibility rules to construct the number

Example: Find the largest 4-digit number divisible by 36. Solution:

  • 36 = 4 × 9, so need divisibility by both 4 and 9
  • For 4: Last 2 digits must be divisible by 4
  • For 9: Sum of digits must be divisible by 9
  • Start with 9999, work backwards: 9996 ✓

Type 2: Count numbers with given divisibility#

Strategy: Use inclusion-exclusion principle

Example: How many numbers from 1 to 100 are divisible by 2 or 3? Solution:

  • Divisible by 2: 50 numbers
  • Divisible by 3: 33 numbers
  • Divisible by both (6): 16 numbers
  • Answer: 50 + 33 - 16 = 67

Type 3: Find digits that make a number divisible#

Strategy: Use divisibility rules to find constraints

Example: Find all digits $d$ such that $123d4$ is divisible by 3. Solution:

  • Sum = 1 + 2 + 3 + d + 4 = 10 + d
  • Need 10 + d divisible by 3
  • d = 2, 5, 8

⚡ Speed Techniques#

Quick Checks#

  1. Even numbers: Always divisible by 2
  2. Numbers ending in 0 or 5: Always divisible by 5
  3. Numbers ending in 0: Always divisible by 10
  4. Sum of digits = 9: Always divisible by 9

Pattern Recognition#

  1. Alternating patterns: Look for 11, 101, 1001, etc.
  2. Repeating digits: 111, 222, 333, etc. are divisible by 3
  3. Palindromes: Often have special divisibility properties

🚨 Common Mistakes to Avoid#

  1. Confusing 3 and 9 rules: Both use sum of digits, but different requirements
  2. Wrong alternating sum: Remember to alternate signs starting with +
  3. Forgetting to check both conditions: For 6, need both 2 and 3
  4. Calculation errors: Double-check your arithmetic
