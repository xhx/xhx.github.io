Wilson's Theorem

Wilson's Theorem and its applications in AMC 10/12 number theory
January 1, 2024 · 2 min · Xiaohui Xie

Wilson’s Theorem#

Wilson’s Theorem is a fundamental result in number theory that provides a necessary and sufficient condition for a positive integer to be prime.

Statement#

A positive integer $p > 1$ is prime if and only if:

$$(p-1)! \equiv -1 \pmod{p}$$

Examples#

Prime Case#

For $p = 5$ (prime): $$(5-1)! = 4! = 24 \equiv -1 \pmod{5}$$

Composite Case#

For $p = 6$ (composite): $$(6-1)! = 5! = 120 \equiv 0 \pmod{6}$$

Proof Sketch#

If $p$ is prime:#

  • The non-zero elements modulo $p$ form a group under multiplication
  • Each element has a unique inverse
  • The only self-inverse elements are $1$ and $p-1$
  • Pairing other elements with their inverses gives $(p-1)! \equiv 1 \cdot (p-1) \equiv -1 \pmod{p}$

If $p$ is composite:#

  • If $p$ has a prime factor $q$ with $q^2 \mid p$, then $q$ and $2q$ both appear in $(p-1)!$, so $(p-1)! \equiv 0 \pmod{p}$
  • If $p = q_1 q_2$ with $q_1 \neq q_2$, then both $q_1$ and $q_2$ appear in $(p-1)!$, so $(p-1)! \equiv 0 \pmod{p}$

Applications in AMC Problems#

Primality Testing#

Problem: Is 17 prime?

Solution: Check $(17-1)! = 16! \pmod{17}$

  • This is computationally intensive, but the theorem provides a theoretical test

Finding Remainders#

Problem: Find $(p-1)! \pmod{p}$ for prime $p$

Solution: By Wilson’s Theorem, $(p-1)! \equiv -1 \pmod{p}$

Modular Arithmetic#

Problem: Find $18! \pmod{19}$

Solution: Since 19 is prime, $18! \equiv -1 \pmod{19}$

Corollaries#

For Composite Numbers#

If $n > 4$ is composite, then $(n-1)! \equiv 0 \pmod{n}$

For Prime Powers#

For prime $p$ and integer $k \geq 1$: $$(p^k-1)! \equiv -1 \pmod{p^k}$$

Key Properties#

  1. Necessary and Sufficient: The theorem provides both directions of the implication
  2. Computational Limitation: While theoretically elegant, it’s not practical for large numbers
  3. Historical Significance: One of the oldest known primality tests
  4. Group Theory Connection: The proof relies on properties of finite groups

Common Mistakes#

  • Forgetting the condition: The theorem only applies to integers $p > 1$
  • Wrong direction: Remember it’s “if and only if”
  • Computational errors: Be careful with large factorials
