Problem 4#

Tags: First Pass 路 source: AMC12 2020 #4

What is the remainder when $2^{10}$ is divided by 7?

A) $1$
B) $2$
C) $3$
D) $4$
E) $5$

Strategic Analysis

First Pass: Problem #4, manageable. Use modular arithmetic: $2^3 = 8 \equiv 1 \pmod{7}$, so $2^{10} = 2^{3 \cdot 3 + 1} \equiv 2 \pmod{7}$.

Time Target: 1 minute

Answer & Solution

Answer: B

Since $2^3 = 8 \equiv 1 \pmod{7}$, we have $2^{10} = 2^{3 \cdot 3 + 1} \equiv 2 \pmod{7}$.

Problem 5#

Tags: First Pass 路 source: AMC12 2020 #5

A fair coin is flipped 3 times. What is the probability of getting exactly 2 heads?

A) $\frac{1}{8}$
B) $\frac{1}{4}$
C) $\frac{3}{8}$
D) $\frac{1}{2}$
E) $\frac{3}{4}$

Strategic Analysis

First Pass: Problem #5, manageable. There are $2^3 = 8$ total outcomes. Getting exactly 2 heads can happen in $\binom{3}{2} = 3$ ways: HHT, HTH, THH. So the probability is $\frac{3}{8}$.

Time Target: 1 minute

Answer & Solution

Answer: C

There are $2^3 = 8$ total outcomes. Getting exactly 2 heads can happen in $\binom{3}{2} = 3$ ways: HHT, HTH, THH. So the probability is $\frac{3}{8}$.
