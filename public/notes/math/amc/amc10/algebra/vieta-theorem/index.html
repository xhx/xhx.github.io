Vieta’s theorem#

Vieta’s theorem relates the coefficients of a polynomial to sums and products of its roots. It’s especially useful for quadratic, cubic, and higher-order polynomials. Here’s the theorem for different polynomial degrees:

Quadratic Equation#

For a quadratic equation ( ax^2 + bx + c = 0 ), with roots ( r_1 ) and ( r_2 ), Vieta’s relations are:

  • ( r_1 + r_2 = -\frac{b}{a} ) (sum of the roots)
  • ( r_1 \cdot r_2 = \frac{c}{a} ) (product of the roots)

Cubic Equation#

For a cubic equation ( ax^3 + bx^2 + cx + d = 0 ), with roots ( r_1 ), ( r_2 ), and ( r_3 ), Vieta’s relations are:

  • ( r_1 + r_2 + r_3 = -\frac{b}{a} )
  • ( r_1r_2 + r_1r_3 + r_2r_3 = \frac{c}{a} )
  • ( r_1r_2r_3 = -\frac{d}{a} )

General Polynomial#

For a polynomial of degree ( n ): [ a_nx^n + a_{n-1}x^{n-1} + \dots + a_1x + a_0 = 0 ] with roots ( r_1, r_2, \dots, r_n ), Vieta’s theorem expresses:

  • The sum of the roots (taken one at a time) as ( -(a_{n-1}/a_n) )
  • The sum of the products of the roots taken two at a time as ( a_{n-2}/a_n )
  • And so on, until the product of all roots is ( (-1)^n \cdot (a_0/a_n) ).

These relations help solve polynomials without explicitly finding the roots.
