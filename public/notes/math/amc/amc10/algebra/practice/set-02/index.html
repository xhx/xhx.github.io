E) $9$

Answer & Solution

Let the roots be $r$ and $s$. We have $r + s = 6$ and $|r - s| = 2$. Since $r + s = 6$ and $r - s = 2$, we get $2r = 8$ and $r = 4$, so $s = 2$. Therefore $q = rs = 4 \cdot 2 = 8$.

Answer: B

Multiplying by $x(x-1)(x+1)$: $x(x+1) + x(x-1) = 2(x-1)(x+1)$, so $x^2 + x + x^2 - x = 2(x^2 - 1)$, giving $2x^2 = 2x^2 - 2$, which simplifies to $0 = -2$. This is impossible, so there are no solutions. The sum is $0$.

If $3^x = 4^y = 12^z$, what is $\frac{1}{x} + \frac{1}{y}$ in terms of $z$?

A) $\frac{1}{z}$
B) $\frac{2}{z}$
C) $\frac{3}{z}$
D) $\frac{1}{2z}$
E) $\frac{1}{3z}$

Answer & Solution

Answer: A

Let $3^x = 4^y = 12^z = k$. Then $x = \log_3 k$, $y = \log_4 k$, and $z = \log_{12} k$. We have $\frac{1}{x} + \frac{1}{y} = \frac{1}{\log_3 k} + \frac{1}{\log_4 k} = \log_k 3 + \log_k 4 = \log_k(3 \cdot 4) = \log_k 12 = \frac{1}{\log_{12} k} = \frac{1}{z}$.

How many solutions does $|x-2| + |x-4| = 3$ have?

A) $0$
B) $1$
C) $2$
D) $3$
E) Infinitely many

Answer & Solution

Answer: C

For $x \leq 2$: $|x-2| + |x-4| = (2-x) + (4-x) = 6-2x = 3$, so $x = \frac{3}{2}$. For $2 < x < 4$: $|x-2| + |x-4| = (x-2) + (4-x) = 2$, which is not equal to 3. For $x \geq 4$: $|x-2| + |x-4| = (x-2) + (x-4) = 2x-6 = 3$, so $x = \frac{9}{2}$. Therefore, there are 2 solutions: $x = \frac{3}{2}$ and $x = \frac{9}{2}$.
